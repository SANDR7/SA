## Begin with:

for next.js
```bash
cd nextjs/
# then
npm install
# then
npm run dev
```

for sanity
```bash
cd sanity/
# then
npx sanity start

# when done
npx sanity deploy || undeploy
# or
npx sanity graphql deploy || undeploy
```
- [Sanity Documentation](https://www.sanity.io/docs/introduction/getting-started?utm_source=readme)
